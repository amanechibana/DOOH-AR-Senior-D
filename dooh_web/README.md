# DOOH Web - React Application

A React-based web application for real-time building detection using YOLO ONNX models in the browser.

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Ensure the model file is present:**
   - The ONNX model should be at: `public/`
   - If missing, copy it from `../dooh_ai/models/`

## Running the Application

### Development Mode
```bash
npm run dev
```

The app will automatically open at `http://localhost:8000` in your browser.
